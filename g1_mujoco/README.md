## Visualization with [MuJoCo](https://github.com/google-deepmind/mujoco)

1. Open MuJoCo

   ```bash
   cd humanoid-playground/g1_mujoco/bin
   ./simulate
   ```

2. Drag and drop the MJCF/URDF model file (`g1_XXX.xml`/`g1_XXX.urdf`) to the MuJoCo Viewer.
